# ✈️ Tableau-British-Airways-Dashboard

An interactive Tableau dashboard analyzing **British Airways customer reviews** across countries, aircraft types, traveler types, and time. Includes dynamic metric selector, map filtering, aircraft comparison, and trend analysis.  
\
**View Online (No Download Required)**: [British Airways Dashboard on Tableau Public](https://public.tableau.com/app/profile/michellefangg/viz/BritishAirwaysReview_17591932335800/Dashboard1)  
**Download**: [British Airway Reviews.twbx](https://github.com/michellefangg/British-Airways-Tableau-Dashboard/raw/main/British%20Airways%20Reviews.twbx)

### Dataset
This dashboard is based on publicly available **British Airways review data**, including ratings for cabin service, food, entertainment, seating, value for money, and overall experience.  
Each review record includes aircraft type, seat class, traveler type, origin country, and review date.  


### Dashboard Interactivity
- **Metric Picker** → dynamically switch between rating categories such as Cabin Staff, Entertainment, Food & Beverages, Ground Service, Seat Comfort, etc.  
- **Date Filter** → analyze trends over time via a continuous monthly slider  
- **Seat Type Filter** → Business, Economy, First Class, Premium Economy  
- **Traveler Type Filter** → Business, Couple, Family, Solo Leisure  
- **Aircraft Filter** → aircraft models with >50 reviews shown individually, others grouped  
- **Continent Filter** → region-based filtering mapped from reviewer country  
- **Cross-Filtering** → clicking a country, aircraft, or month updates all visuals  


### Features & Visuals
- Geographic map of **average ratings by country**  
- Monthly trend line for selected rating metric  
- Bar chart of **aircraft type vs. average rating + review count**  
- KPI summary cards showing overall average scores  
- Fully interactive dashboard with Tableau actions + filters  
- Custom color palette (green + pink), formatted tooltips, 1-decimal precision
